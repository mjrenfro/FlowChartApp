__*Simple Mongo Setup Reminder*__

* Get Mongo server running
 * sudo mongod

* Shell Running
 * mongo

__*Starting NodeJS App*__

* node server.js

* *If want a easy way to restart app after changes*

* npm install nodemon

* nodemon server
